# Telegram-bot "Магазин 3D-принтеров"

## Описание

Бот представляет собой удобную платформу для заказа 3D-принтеров различных брендов. Пользователи могут просматривать каталог, добавлять товары в корзину и оформлять заказы непосредственно через Telegram.

**Основные плюсы:**
- Удобный каталог товаров 
- Простая система корзины
- Автоматическое формирование заказов
- Панель управления для администраторов

## Основные функции

### **Просмотр товаров**
- **Каталог продукции** - просмотр продукции с указанием модели и цены за товар
- **Выбор** - удобный подбор товаров с помощью простого кнопочного интерфейса

### **Работа с корзиной**
- **Добавление товаров** - выбор количества для каждого товара
- **Просмотр корзины** - список выбранных товаров с количеством
- **Редактирование** - изменение количества или удаление позиций
- **Очистка корзины** - полное удаление всех товаров

### **Оформление заказа**
- **Автоматический расчет** - подсчет общей суммы заказа
- **Формирование чека** - создание Excel-файла с детализацией заказа
- **Уведомление** - автоматическая отправка заказа администратору
- **Подтверждение пользователю** - отправка чека покупателю

### **Личный кабинет**
- **Удобный доступ** к корзине и истории заказов
- **Управление профилем** в одном месте

### **Административные функции**
- **Загрузка товаров** - обновление каталога через Excel-файлы
- **Выгрузка отчетов** - получение данных о заказах
- **Управление базой**

## Инструкция по использованию

### 1. **Начало работы**
- Отправьте команду `/start`
- Ознакомьтесь с приветственным сообщением
- Выберите нужный раздел из главного меню

### 2. **Просмотр товаров**
- Нажмите кнопку **"Общая выпускаемая продукция"**
- Выберите интересующий товар из списка
- Нажмите на кнопку соответсвующую интересующему вас товару

### 3. **Добавление в корзину**
- Введите количество товара
- Товар автоматически добавится в корзину

### 4. **Управление корзиной**
- Перейдите в **"Личный кабинет"** → **"Корзина"**
- Просмотрите выбранные товары
- При необходимости:
  - Измените количество
  - Удалите ненужные позиции
  - Очистите всю корзину

### 5. **Оформление заказа**
- В корзине нажмите **"Оплатить товары"**
- Система автоматически:
  - Сформирует детализированный чек
  - Отправит заказ администратору
  - Предоставит вам копию чека
  - Очистит корзину

## Панель администратора

### Доступ к панели
- Команда `/admin_panel`
- Доступ только для авторизованных администраторов

### Функции администратора

#### **Загрузка товаров**
- Подготовьте Excel-файл с колонками:
  - **Название товара**
  - **Цена** (в рублях)
- Отправьте файл боту
- Система автоматически обновит каталог

#### **Выгрузка отчетов**
- Получение статистики по заказам
- Анализ популярности товаров
- Управление базой данных


## Команды бота

- `/start` - запуск бота и главное меню
- `/admin_panel` - панель администратора (только для админов)
